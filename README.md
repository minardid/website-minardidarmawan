# Minardi Darmawan - Personal Portfolio Website

## Overview
Welcome to my personal portfolio website! This responsive website showcases my professional journey, skills, and projects. It serves as a digital resume and a way to connect with potential collaborators and employers.

## Features

- **Professional Profile**: Overview of my experience and expertise
- **Interactive Resume**: Detailed work history and education
- **Portfolio Showcase**: Highlighting key projects and achievements
- **Contact Form**: Easy way to get in touch directly via WhatsApp
- **Responsive Design**: Optimized for all device sizes
- **Modern UI**: Clean and professional interface with smooth animations

## Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript (jQuery)
- **Styling**: Custom CSS with animations
- **Form Handling**: Custom JavaScript validation
- **Hosting**: Firebase Hosting
- **Version Control**: Git & GitHub

## Deployment

The website is automatically deployed to Firebase Hosting. Any push to the `main` branch triggers a new deployment.

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/minardid/website-minardidarmawan.git
   ```

2. Open `index.html` in your browser to view the website locally.
